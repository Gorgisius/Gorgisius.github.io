HTML and CSS

Hyper Text Markup Language and Cascading Style Sheets are the two basic languages used to create static webpages that display information in an aesthetic way. In this issue of WDYKA we'll be teaching you the basic syntax and words used by HTML and CSS to make pretty and useful websites. Lets begin!

HTML

What is HTML?

Hyper Text Markup Language is a standard markup language (duh) that uses tags and attributes to create the structure or the "skeleton" of a website. If you were to create a website using only HTML, it would look like this:

Not very appealing right? That's why it is always used alongside CSS (We will get to CSS in a minute).

HTML tags & elements

HTML tags are the building blocks used by HTML that tell our web browsers how to structure and display the content that is in them. They are enclosed in angle brackets (< >) and can be opening (<h1>) or closing (</h1>). There are also self-closing tags like <img> which is used to add images.

An HTML element is composed of a starting tag, some content, and an end tag:

<h1>This is a heading</h1>

The HTML element is everything from the start tag to the end tag.

HTML's most used tags

Here is a table with some of the most used HTML tags and their meanings:

Tag Meaning
<html> Represents the root of an html document
<h1/2/3/4/5/6> Headers ranging from biggest (1) to smallest (6)
<p> Defines a paragraph
<ol> Defines an ordered list
<ul> Defines an unordered list
<li> Defines a list item inside a list
<head> Contains the metadata of the webpage
<body> Contains the contents of the webpage
<footer> Defines a footer
<div> Defines a division in an html document
<img> Used to embed an image
<video> Used to embed a video

CSS

What is CSS?

Cascading Style Sheets is a style sheet language (duh) that uses rules, selectors and declarations to add style to a webpage written using a markup language like HTML.

Together, HTML and CSS make the perfect duo.

CSS rules, selectors, declarations, properties and values

CSS is a rule-based language where you specify styles that should be applied to elements in your webpage.

A CSS rule is made up of a selector and declarations.


Selector ---> h1 {
Declarations --------> color: black;
                            font-weight: bold;
            }

Declarations are made up of a property and a value.


Property --> color: black; <-- Value

The above rule selects all the h1 elements and makes their text black and bold.

How to use HTML and CSS together?

In order to add CSS to your HTML you will have to link your '.css' file to your '.html' file using the amazing <link> tag inside of the <head> definition.

Like this:


<link> rel="stylesheet" href="relative_path_to_your_css_file" </link>

Now you can make your website look amazing.
